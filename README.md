# Advanced Web3 React DApp (Sepolia + Multi-Wallet)

Ushbu loyiha React.js va eng so'nggi Web3 kutubxonalari (**Wagmi**, **RainbowKit**, **Viem**) yordamida yaratilgan zamonaviy Blokcheyn ilovasi hisoblanadi.

## 🌟 Yangi Xususiyatlar
- **Framework**: React.js (Vite) bilan to'liq qayta yozildi.
- **Multi-Wallet Support**: 
  - MetaMask
  - Coinbase Wallet
  - WalletConnect (Ko'plab mobil hamyonlar uchun)
- **Transaction Status UI**:
  - `Pending`: Blokcheynda tasdiqlanish jarayoni.
  - `Success`: Muvaffaqiyatli tranzaksiya animatsiyasi.
  - `Error`: Xatoliklarni aqlli boshqarish.
- **Premium Interfeys**: Tailwind CSS va Glassmorphism dizayni.
- **Testnet**: Sepolia tarmog'i bilan integratsiya qilingan.

## 🛠 O'rnatish (Local)
1. Repository-ni klon qiling.
2. `npm install` buyrug'ini bering.
3. `npm run dev` orqali mahalliy serverni ishga tushiring.

## 🌐 Deploy
Loyiha avtomatik ravishda GitHub Pages yoki Vercel-ga joylash uchun tayyorlangan (`vite.config.js` sozlangan).
